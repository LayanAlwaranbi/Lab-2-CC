# Machine Learning Lab 2  
## Customer Churn Classification

### Problem Description
The objective of this project is to predict whether a customer will churn (leave the service) or not based on customer information and service usage data. This is a classification problem where the target variable is Churn, which has two possible values: Yes or No.

### Dataset Description
The dataset used in this project is the Telco Customer Churn Dataset, which is a publicly available tabular dataset. It contains customer demographic information, account details, and service usage data. The dataset includes 7,043 records and 21 attributes, with the target variable being the Churn column.

### Machine Learning Problem Type
- Type: Classification  
- Target Variable: Churn (Yes / No)

### Tools Used
- Python  
- Pandas  
- Google Colab
